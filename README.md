# PowerAuto AI Core

基于三层架构哲学的智能AI核心系统

## 架构原则

- **最小前置，最大自进化**
- **搜索 + 工具 = 完事**
- **三层架构：搜索 → MCP → Kilo Code兜底**
- **所有需求都在MCP体现，其他人照做**

## 核心特性

- 🔍 智能搜索引擎
- 🔧 标准化MCP工具集
- 🛡️ Kilo Code兜底机制
- 🤖 智能介入系统
- 📱 跨平台支持

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 启动AI Core
python aicore.py

# 运行智能介入
python smart_intervention.py
```
